Soy programador web freelancer desde el año 2004. Comencé programando sitios web enteramente en Flash, era muy común en ese tiempo ver sitios web cargados de animaciones y sonidos.

Luego en 2006 me especialicé en maquetación web, convirtiendo sitios desde PSD a xHTML con validación W3C. Con el tiempo comencé el desarrollo de sitios web estáticos (2010) escribiendo el CSS, Javascript y HTML dominando rápidamente a foundation de Zurb como framework de desarrollo adaptable (responsive web design) y jQuery.

En el año 2012 inicié la programación web backend con PHP desarrollando sitios basados en Joomla y creando componentes, temas y módulos para ese CMS. Programé durante mucho tiempo sitios en Joomla 2.5 y PHP 5.x y aún ahora trabajo activamente con Joomla 3.x y PHP 7.x.

Mis proyectos más recientes involucran desarrollo en PHP 7.x y el uso de la herramienta Yii2 como framework de desarrollo avanzado al igual que Python 3.x usando Django como framework de desarrollo. He iniciado estudios en C#.
